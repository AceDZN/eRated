# eRated Widget
just trying to make a copy of the eRated seller Widget

### Checkout

Checkout this repo, install dependencies, then start the http-server process with the following:

```
	> git clone git@github.com:AceDZN/eRatedWidget.git
	> cd eRatedWidget
	> npm install
	> gulp
```
